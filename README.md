# 🤖 AI System Architect

AI-powered multi-agent system designer that generates complete architecture flowcharts for any task.

## ✨ Features
- Generate complete multi-agent AI system blueprints
- Visual flowcharts with agent roles, routing rules & feedback loops
- Failure handling & scalability strategies
- Powered by Claude AI

## 🚀 Setup

### 1. Clone the repo
```bash
git clone https://github.com/bardhan730-jpg/ai-system-architect.git
cd ai-system-architect
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Add your API key
```bash
cp .env.example .env
# .env file mein apni Anthropic API key daalo
```

### 4. Run the app
```bash
python app.py
```
